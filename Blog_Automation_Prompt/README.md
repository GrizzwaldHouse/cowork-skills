# Blog Automation Prompt

A ready-to-use prompt template for generating blog posts with Claude.

## Purpose

This folder contains a structured prompt template (`prompt_template.md`) that helps you consistently produce high-quality blog posts. Fill in the placeholders, submit the prompt to Claude, and get a well-formatted blog post in return.

## How to Use

1. Open `prompt_template.md`.
2. Copy the prompt section.
3. Replace all `[placeholder]` values with your specific content.
4. Submit the completed prompt to Claude.
5. Review and edit the generated post before publishing.

## Tips

- Be specific with your key points -- the more detail you provide, the better the output.
- Use the SEO keywords field to ensure the post targets the right search terms.
- For technical blogs, always request code examples and step-by-step instructions.
