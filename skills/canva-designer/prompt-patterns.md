# Canva Prompt Patterns — Query Templates by Design Type

## Logo

### Template
```
Create a [style] [mark type] logo for "[brand name]", a [industry/description].
Color palette: [primary color] with [accent color] accent on [background].
Style: [characteristics]. The logo should [scalability requirement].
```

### Example
```
Create a minimalist combination mark logo for "TechFlow", a B2B SaaS analytics platform.
Color palette: deep navy (#1E3A5F) with electric blue (#00B4D8) accent on dark background.
Style: modern, clean, geometric shapes, tech-forward. The logo should work from 32px favicon to large format print.
```

### Key Variables
- **Mark types**: wordmark (text only), lettermark (initials), icon mark (symbol only), combination mark (icon + text)
- **Style words**: minimalist, bold, elegant, playful, geometric, organic, vintage, futuristic
- **Always specify**: scalability, background compatibility (light AND dark), industry context

## Social Media Post

### Template
```
Design a [platform] post for [brand/purpose]. Style: [aesthetic]. Colors: [palette].
Headline: "[text]" displayed prominently. Supporting text: "[text]".
Layout: [image/text position]. Mood: [tone]. Include [CTA/element].
With generous whitespace and clean, modern composition.
```

### Example
```
Design an Instagram post for a product launch announcement. Style: bold and modern with gradient accents.
Colors: violet (#8B5CF6) to cyan (#06B6D4) gradient with white text.
Headline: "Introducing Cowork" displayed prominently center. Supporting text: "AI-powered desktop agent for Windows".
Layout: centered text over abstract gradient background. Mood: exciting, tech-forward.
Include a subtle "Learn more" CTA at bottom. With generous whitespace.
```

### Platform-Specific Notes
- **Instagram**: Bold visuals, minimal text (keep under 20% of image area), vibrant colors
- **LinkedIn**: Professional, clean, data-driven visuals welcome, more text acceptable
- **X/Twitter**: High contrast, readable at small size in feed, eye-catching in 1 second
- **Facebook**: Warm, engaging, avoid text-heavy (algorithm penalizes >20% text)

## Presentation

### Outline Review Template (for request-outline-review)
- **topic**: "[Specific Title] — [Context/Purpose]"
- **audience**: "[Role] [knowledge level] at [organization type]"
- **style**: One of: minimalist, elegant, digital, geometric, playful, modular, organic
- **length**: "short" (executive summary, 1-5 slides) or "balanced" (standard, 5-15 slides)

### Slide Description Patterns (max 90 chars each)
- Title slide: "Overview of [topic] for [audience] with key takeaways"
- Problem slide: "[Problem statement] and its impact on [stakeholder]"
- Solution slide: "How [solution] addresses [problem] with [key benefit]"
- Data slide: "[Metric] showing [trend/insight] across [time/segments]"
- CTA slide: "Recommended next steps and timeline for implementation"

## Poster / Flyer

### Template
```
Design a [size] [poster/flyer] for [event/product/announcement].
Headline: "[text]" as the dominant visual element.
Key details: [date, location, price, URL]. Call to action: "[CTA text]".
Style: [aesthetic] with [color palette]. Most important element: [headline/image/CTA].
Layout: [composition direction]. Include clear visual hierarchy with generous whitespace.
```

### Example
```
Design a portrait poster for a tech conference called "DevSummit 2026".
Headline: "DevSummit 2026" as the dominant visual element.
Key details: March 15-17, San Francisco, Register at devsummit.dev. Call to action: "Register Now".
Style: bold geometric with dark navy (#1E293B) background, electric blue (#4361EE) accents, white text.
Most important element: headline. Layout: Z-pattern composition with logo top-left, headline center, details bottom.
Include clear visual hierarchy with generous whitespace.
```

## Infographic

### Template
```
Create a [topic] infographic presenting [number] key data points.
Data: [list main statistics/facts]. Style: [flat/isometric/illustrated].
Colors: [palette]. Flow: [top-to-bottom/left-to-right/sectioned].
Emphasize: [most important statistic]. Make data comparisons visual with [charts/icons/numbers].
Clean layout with clear section separation and generous whitespace.
```

### Example
```
Create a "State of AI in 2026" infographic presenting 6 key data points.
Data: 78% of companies use AI, $180B market size, 3x productivity gain, 45% cost reduction,
12M new AI jobs, 92% executive priority. Style: flat modern with clean icons.
Colors: violet (#8B5CF6) primary, cyan (#06B6D4) secondary, dark background (#171717).
Flow: top-to-bottom with 3 rows of 2 stats each. Emphasize: 78% adoption rate as hero number.
Make data comparisons visual with large numbers and progress bars.
```

## Dashboard Mockup

### Template
```
Design a [application type] dashboard mockup showing [key sections].
Style: [dark/light] theme, modern flat UI.
Left sidebar: [navigation items]. Top bar: [elements].
Main area: [widgets/content]. Colors: [palette].
Include realistic UI elements with proper spacing and hierarchy.
```

## Universal Quality Boosters

Add any of these phrases to improve ANY Canva query:
- "with generous whitespace and clean layout"
- "professional quality suitable for [context]"
- "with clear visual hierarchy guiding the eye"
- "modern and polished, avoiding visual clutter"
- "balanced composition with intentional spacing"
