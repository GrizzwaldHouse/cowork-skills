# Typography Reference — Font Pairings & Type Scale

## Proven Font Pairings

### Sans-Serif + Sans-Serif (Modern / Clean)
| Heading | Body | Vibe |
|---------|------|------|
| Inter 700 | Inter 400 | Neutral, systematic, developer-friendly |
| Montserrat 600 | Open Sans 400 | Geometric warmth, highly legible |
| Raleway 700 | Lato 400 | Elegant with personality |
| Poppins 600 | Nunito 400 | Rounded, friendly, approachable |
| DM Sans 700 | DM Sans 400 | Compact, modern, geometric |
| Geist 600 | Geist 400 | Vercel-style, tech-forward |

### Serif + Sans-Serif (Editorial / Premium)
| Heading | Body | Vibe |
|---------|------|------|
| Playfair Display 700 | Source Sans Pro 400 | High contrast, editorial luxury |
| Merriweather 700 | Roboto 400 | Warm authority, highly readable |
| Lora 600 | Open Sans 400 | Friendly serif, balanced |
| DM Serif Display 400 | DM Sans 400 | Matched family, sharp contrast |
| Fraunces 700 | Inter 400 | Playful serif, modern body |

### Sans-Serif Heading + Serif Body (Academic / Thoughtful)
| Heading | Body | Vibe |
|---------|------|------|
| Montserrat 600 | Merriweather 400 | Bold geometric + warm reading |
| Poppins 600 | Lora 400 | Rounded modern + classic reading |
| Inter 700 | Source Serif Pro 400 | System clean + editorial body |

### Monospace (Tech/Code Accent Only)
- JetBrains Mono — best for code blocks and data
- Fira Code — ligatures for code, good for UI data elements
- IBM Plex Mono — pairs well with IBM Plex Sans
- **Rule**: Only use monospace for code snippets, data values, or technical labels — never for body text or headings

## Font Weight Reference
| Weight | Name | Usage |
|--------|------|-------|
| 100 | Thin | Decorative display only |
| 200 | Extra Light | Large display headings |
| 300 | Light | Subheadings, large captions |
| 400 | Regular | Body text (default) |
| 500 | Medium | UI labels, emphasized text |
| 600 | Semi Bold | Subheadings, H3-H4 |
| 700 | Bold | Headings H1-H2 |
| 800 | Extra Bold | Display/hero text |
| 900 | Black | Impact/accent only |

## When to Use Which Pairing
| Audience | Recommended Pairing Style |
|----------|---------------------------|
| Corporate / Finance | Sans + Sans (Inter, DM Sans) — clean, trustworthy |
| Editorial / Publishing | Serif heading + Sans body — authoritative yet modern |
| Startup / Tech | Sans + Sans (Poppins, Geist) — friendly, forward |
| Luxury / Fashion | Serif heading + Sans body (Playfair + Source Sans Pro) |
| Academic / Research | Sans heading + Serif body (Montserrat + Merriweather) |
| Developer / Technical | Sans + Monospace accent (Inter + JetBrains Mono) |
