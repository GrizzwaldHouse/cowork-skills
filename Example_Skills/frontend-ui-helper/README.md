# Frontend UI Helper

A Claude Skill for generating and refining front-end UI components.

## Overview

This skill helps you quickly scaffold UI components, fix layout bugs, and implement responsive designs. It supports React, Vue, Svelte, and plain HTML/CSS.

## Quick Start

1. Read `SKILL.md` for the full skill definition and prompt patterns.
2. Describe your desired component to Claude, specifying the framework and any constraints.
3. Integrate the generated code into your project.

## Resources

Place any supporting files (component templates, style guides, design tokens) in the `resources/` folder.

## Related Skills

- `backend-workflow-helper` -- for connecting your UI to back-end APIs
- `game-dev-helper` -- for game-specific UI (HUD, menus)
