# Backend Workflow Helper

A Claude Skill for building and maintaining back-end services, APIs, and data workflows.

## Overview

This skill helps you design APIs, write database queries, implement authentication, set up background jobs, and debug server-side issues. It adapts to your stack and follows your project's conventions.

## Quick Start

1. Read `SKILL.md` for the full skill definition and prompt patterns.
2. Describe the back-end feature or bug you need help with, including your tech stack.
3. Review and integrate the generated code.

## Resources

Place supporting files (API schemas, migration templates, middleware examples) in the `resources/` folder.

## Related Skills

- `frontend-ui-helper` -- for building the UI that consumes your APIs
- `workflow-productivity` -- for automating repetitive back-end tasks
