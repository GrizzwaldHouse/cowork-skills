# Documentation & Blog Generator

A Claude Skill for producing technical documentation, blog posts, and developer guides.

## Overview

This skill generates well-structured written content from code, schemas, commit history, or topic outlines. It supports API docs, tutorials, changelogs, and blog posts in Markdown and other formats.

## Quick Start

1. Read `SKILL.md` for the full skill definition and prompt patterns.
2. Provide your source material and specify the content type, audience, and tone.
3. Review and publish the generated content.

## Resources

Place supporting files (content templates, style guides, example posts) in the `resources/` folder.

## Related Skills

- `workflow-productivity` -- for automating documentation builds and publishing
- `backend-workflow-helper` -- source of API code to document
