# Game Dev Helper

A Claude Skill for game development across Unreal Engine 5, Unity, and Godot.

## Overview

This skill assists with gameplay programming, AI systems, physics, animation, shader code, and engine-specific patterns. It follows each engine's conventions and best practices.

## Quick Start

1. Read `SKILL.md` for the full skill definition and prompt patterns.
2. Describe your game development task, specifying the engine and subsystem.
3. Review the generated code and test it in your engine.

## Resources

Place supporting files (behavior tree templates, shader snippets, configuration presets) in the `resources/` folder.

## Related Skills

- `frontend-ui-helper` -- for game menu and HUD UI design
- `backend-workflow-helper` -- for game server and multiplayer back-end logic
