# Workflow Productivity

A Claude Skill for automating repetitive tasks and improving developer productivity.

## Overview

This skill helps you write automation scripts, configure CI/CD pipelines, manage file operations in bulk, and streamline your daily development workflow.

## Quick Start

1. Read `SKILL.md` for the full skill definition and prompt patterns.
2. Describe the task you want to automate, including your environment and current manual process.
3. Review the generated script or configuration and integrate it.

## Resources

Place supporting files (script templates, pipeline configs, task runner presets) in the `resources/` folder.
